About

Yet Another Football Database is database of divisions, teams and players in the NFL. This website was created as a class project for Glenn Downing's CS373 at the University of Texas. It's populated with data scraped from footballdb.com. Last updated on July 2, 2015.

Data Collection:

Modules Used: re (Regular Expressions), bs4 (Beautiful Soup), requests (Requests).

  • Regular Expressions is used to match class names with information.
  • Beautiful Soup is used to perform searches on the HTML.
  • Requests is used to load HTML content of a URL.



Tools:

  • GUI: Bootstrap
  • Frontend: Javascript and AngularJS
  • Backend: Python and Django
  • Database: PostgreSQL
  • Website Hosted on Rackspace
