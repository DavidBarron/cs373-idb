About Us

Yet Another Football Database is database of divisions, teams and players in the NFL. This website was created as a class project for Glenn Downing's CS373 at the University of Texas. It's populated with data scraped from ESPN NFL and Wikipedia. Last updated on July 21, 2015.

Data Collection:

Modules Used: re (Regular Expressions), bs4 (BeautifulSoup), requests (Requests).

  • Requests is used to load HTML content of a URL.
  • BeautifulSoup is used to perform searches on the HTML passed in from Requests.
  • Regular Expressions is used in conjunction with BeautifulSoup to match segments of class names with information on the page being scraped.
  • Scraping is a simple process of pattern matching.



Tools:

  • GUI: Bootstrap
  • Frontend: Javascript and AngularJS
  • Backend: Python and Django
  • Database: PostgreSQL
  • Website Hosted on Rackspace
